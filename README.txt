ScaleHQ Website (Static)

What's included
- index.html
- styles.css
- script.js
- assets/scalehq-logo.png

How to use
1) Download and unzip this folder.
2) Upload the contents to your web host (Netlify, Vercel static, Cloudflare Pages, cPanel, etc.).
3) Ensure index.html is in the root.

Where to edit
- Copy: edit index.html (search for the sections: HERO, WHO, WHAT, HOW, TEAM, WHY, CTA)
- Colors: edit CSS variables at the top of styles.css
- Links:
  - The "Apply" and "Book" buttons are placeholders. Replace the #apply and #book anchors or change them to your application URL / Calendly.

Optional improvements
- Replace assets/scalehq-logo.png with a clean transparent PNG/SVG for best results.
- Add a favicon (favicon.ico) in the root and update the <link rel="icon"> tag.
